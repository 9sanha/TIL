# 🧐 배운 것 🧐

### 우분투 환경 세팅

``` markdown
>> sudo apt-get update
오픈 jdk 설치
>> sudo apt-get install openjdk-8-jdk
버전확인
>> java -version 
현재 실행중인 pid 값(프로세스 번호) 조회
>> ps -ef | grep java
프로세스 종료
>> kill -9 종료시킬 프로세스 id값
원격접속 종료 후에도 서버 구동되게 하기
>>nohup java -jar 프로젝트명.jar &
```
